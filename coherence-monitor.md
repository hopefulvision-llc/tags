## Implementation Notes
**Purpose**: Real-time consciousness state tracking and biometric feedback

**Activation**: Always visible but **full dashboard** appears at 75%+ coherence

**Visual Design**:
- Icon: 📊 or 👁️ symbol
- Color: Calm blue (#3498DB)
- Layout: Dashboard format with multiple metrics
- Animation: Smooth transitions, gentle pulsing

**Displayed Metrics**:
- **Primary**: Current coherence percentage (0-100%)
- **HRV Graph**: Real-time heart rate variability visualization
- **Breath Rate**: Breaths per minute tracking
- **Coherence Trend**: 5-minute rolling average
- **Session Stats**: Time in high coherence, activation count
- **Status Messages**: Contextual feedback based on state

**State-Based Messages**:
- Below 75%: "Breathe to activate agents..."
- At 75%: "Agents resonating ✨"
- Above 90%: "Deep coherence achieved"
- Peak moments: "Peak coherence achieved" / "You are the witness" / "Sacred observer state activated"

**Technical Architecture**:
```javascript
CoherenceMonitorAgent = {
  state: 'dormant' | 'active' | 'peak',
  displayMode: 'minimal' | 'full-dashboard',
  activationThreshold: 75,
  peakThreshold: 90,
  updateFrequency: 1000 // 1 second
}
```

**Q1 Demo Limitations**:
- Simulated HRV (from breath timing)
- No persistent data storage
- Basic visualization (not production-grade)

## Design Philosophy

**Core Principle:**
Technology is not neutral. It either amplifies human consciousness or extracts from it. There is no middle ground - every design decision moves in one direction or the other.

**Test Questions:**
1. After using this feature, is the user more or less present?
2. Does this increase human agency or decrease it?
3. Would this work without exploiting psychology?
4. Does this make consciousness visible or hidden?
5. Does this serve the user's highest self or their compulsions?

**Decision Framework:**
- If amplifying: Build it
- If extracting: Refuse it
- If unsure: Prototype and measure consciousness impact

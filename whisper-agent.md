## Implementation Notes
**Purpose**: Intuitive insights and gentle guidance through minimal transmission

**Character Limit**: Strict 20-character maximum (including spaces)

**Activation**: Coherence threshold 75%+ triggers agent messages

**Visual Design**:
- Icon: 🌙 Moon or wind symbol
- Color: Soft purple (#9B59B6)
- Animation: Gentle drift like smoke
- Effect: Typing animation (optional)

**Message Examples** (all ≤20 chars):
- "528 Hz flow ✨"
- "You're here now"
- "Breath creates space"
- "Notice the stillness"
- "Coherence is allowing"
- "Invite, don't force"

**Technical Architecture**:
```javascript
WhisperAgent = {
  state: 'dormant' | 'active',
  activationThreshold: 75,
  maxCharacters: 20,
  messageRotation: 10000, // 10 seconds
  characterValidation: strict
```

**Design Rationale**:
1. **Attention economics**: Short messages respect user focus
2. **Cognitive load**: Minimal text prevents overwhelm
3. **Poetic constraint**: Forces precision and resonance
4. **Non-intrusive**: Gentle presence vs demanding interface
